# Streamlitの便利なプラグイン

- [stlite](https://github.com/whitphx/stlite)
    - StreamlitをWebブラウザ上だけで動作さになるせることができる😮！！！
	- Pythonサーバーを立てずに、静的なウェブページを用意するだけで動作する

- [@stlite/desktop](https://github.com/whitphx/stlite/tree/main/packages/desktop)
    - OS上のデスクトップアプリ化ができる😮！！！
      - [Python Web UIフレームワークで作るデスクトップアプリ](https://gihyo.jp/article/2024/04/monthly-python-2404#gh3edsBm0D) の記事に方法の詳細が載っています

- [streamlit-webrtc](https://github.com/whitphx/streamlit-webrtc)
  - ネットワーク経由でのリアルタイムビデオ、オーディオストリームの処理が可能😮！！！
