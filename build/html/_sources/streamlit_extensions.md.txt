# Streamlitの便利なプラグイン

- [stlite](https://github.com/whitphx/stlite)
    - StreamlitをWebブラウザ上だけで動作させることができる😮!!
	- Pythonサーバーを立てずに、静的なウェブページを用意するだけで動作します

- [@stlite/desktop](https://github.com/whitphx/stlite/tree/main/packages/desktop)
    - OS上のデスクトップアプリ化ができる😮!!
      - [Python Web UIフレームワークで作るデスクトップアプリ](https://gihyo.jp/article/2024/04/monthly-python-2404#gh3edsBm0D) にアプリ化に関する詳細が載っています

- [streamlit-webrtc](https://github.com/whitphx/streamlit-webrtc)
  - ネットワーク経由でのリアルタイムビデオ、オーディオストリームの処理が可能😮!!
    - [ブラウザで動くリアルタイム画像/音声処理アプリをStreamlitでサクッと作る](https://zenn.dev/whitphx/articles/streamlit-realtime-cv-app) に使い方に関する詳細が載っています
